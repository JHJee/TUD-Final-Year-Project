Lesson 1
SQL / NoSQL Injection


SQL injection is inserting SQL statements into an input field in order to execute the statements and get access to the database contents against the entry's intended purpose.
This attack can be done when the form processes the raw inputs from the user input to retrieve data from the database. And can be prevented in a lot of ways.
PDO(PHP), ORM(Django), etc.
